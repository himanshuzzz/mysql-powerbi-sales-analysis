# MySQL + Power BI Sales Analysis Project

## Project Overview
This project analyzes sales data using **MySQL** for data processing and **Power BI** for interactive visualization.

## Tools & Technologies
- MySQL
- Power BI
- SQL
- CSV Data

## Key Analysis
- Revenue trends by customers
- Top-performing products
- Monthly sales performance
- Yearly distribution of records

## Database Design
- Tables imported into MySQL
- Data cleaned using SQL queries
- Aggregations created for Power BI
